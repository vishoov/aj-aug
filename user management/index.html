User Management API

This is a simple user management API built with Node.js and Express.

Endpoints

  • GET /users: Retrieve a list of all users.
  • GET /users/:id: Retrieve a specific user by ID.
  • POST /users: Create a new user. Requires a JSON body with user data
  • PUT /users/:id: Update an existing user by ID. Requires a JSON body with "name" and/or "email".
  • DELETE /users/:id: Delete a user by ID.

Example Requests

GET /users
Response:
[
    {"id": 1, "name": "John Doe", "email": "abc@gmail.com"},
    {"id": 2, "name": "Jane Smith", "email": "haha@gmail.com"}
]
        
GET /users/1
Response:
{"id": 1, "name": "John Doe", "email": "abc@gmail.com"}
        
POST /users
Request Body:
{
    "name": "Alice Johnson",
    "email": "abc@gmail.com",
    "password": "password123"
}
Response:
{"id": 3, "name": "Alice Johnson", "email": "abc@gmail.com"}
        
PUT /users/1
Request Body:
{
    "name": "John Updated",
    "email": "updated@gmail.com"
}
Response:
{"id": 1, "name": "John Updated", "email": "updated@gmail.com"}
        
DELETE /users/1
Response:
{"message": "User with ID 1 has been deleted."}
